# .config

## Description

Contains configs for `~/.config/`.  
Will get extended over time.  
I'm trying to make this smart but we'll see.

# Usage

Clone this repo wherever you have `$XDG_CONFIG_HOME` set i guess.

To use aliases:
```sh
# .bashrc / .zshrc
source $XDG_CONFIG_HOME/.shared/aliases.sh
```
